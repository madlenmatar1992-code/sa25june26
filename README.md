# Patient Sentiment Dashboard

Static sentiment dashboard for patient feedback, ready for Netlify deployment.

## Live App

The app entrypoint is:

- `sentiment analysis v3/index.html`

The dashboard uses:

- `sentiment analysis v3/styles.css`
- `sentiment analysis v3/script.js`
- `sentiment analysis v3/patient_sentiment_dashboard_data.js`

## Netlify Deployment

1. Connect this GitHub repository to Netlify.
2. Set the publish directory to:

   `sentiment analysis v3`

3. Deploy the site.

## Local Preview

Open `sentiment analysis v3/index.html` in a browser, or serve the folder with any static file server.

## Data Source

The dashboard is populated from:

- `sentiment analysis v3/Sentiment Analysis - Test 4 - Classified.xlsx`
- sheet 2: `Analyzed Data`
